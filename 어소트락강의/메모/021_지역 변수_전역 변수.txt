지역변수와 전역변수가 있음

함수안에 쓰면 지역변수
함수 밖에 쓰면 전역변수

전역변수는 데이터영역을 사용

변수의 종류
1. 지역변수
2. 전역변수
3. 정적 변수
4. 외부변수

메모리 영역
1. 스택영역
2. 데이터 영역
3. 읽기 전용(코드, ROM)
4. 힙 영역

메모리 영역은 OS마다 또 세부적으로 따지면 더 많다
하지만 대표적으로는 위처럼 되었있다.

C++은 메모리영역에 대한 개념을 자세히 알고 있어야 한다.
그리고 자신이 작성한 코드가 메모리를 어떤식으로 사용하고 돌아가고 있는지 명확하게 파악해야 한다.
대신 숙달이 된다면 자기가 시간만 추가하면 어떤것이든 만들 수 있을 것이다.

프로그램의 진입점 함수인 Main함수가 호출되면서
즉 프로그램이 시작과 동시에 데이터영역이 만들어진다.
그래서 데이터영역을 사용하는 전역변수는 프로그램 시작과 동시에 자리를 차지한다.
그리고 프로그램이 종료될 때 해제과 된다.

즉 프로그램이 실행되는 내내 존재한다.

Data 영역 특징
프로그램 시작 시 생성
프로그램 종료 시 해제

지역변수는
void Test()
{
	int i = 0; // 지역변수
	++i;
}
만들고
Test(); 호출하였을 때
여러번 호출해도 i는 함수가 끝날 때 해제되기 때문에
i값은 늘어 나지 않는다

하지만
int g_i = 0;

void Test()
{
	++g_i; // 전역변수
}
이런 함수라면
Test()를 3번부르면
g_i는 3이 되어 있을 것이다.

지역변수는 함수가 끝나면 스텍영역 해제가 되지만
전역변수는 데이터변수에서 끝까지 있다.

그래서 프로그램이 실행되는 동안  없어지면 안되는 것들을 올려놓으면 된다.

또 그래서 지역변수는 선언된 함수에서만 접근이 가능하지만 
전역변수는 모든 곳에 접근이 가능하다.
