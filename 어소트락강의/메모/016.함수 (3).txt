팩토리얼 함수 만들기
4팩토리얼을 한다면 1 * 2 * 3 * 4 즉 3번을 곱하기 때문에
i의 값보다 1번 적게 반복문을 돌리면 된다.

int i = 4;
int iValue = 1;

for (int j = 0; j < i - 1; ++j) 
{
	iValue *= (j + 2);
}
printf("%d", iValue);

이 코드는 함수로 구현하지 않았다 하지만 다른 코드에 이 코드가 쓰일 때
이 코드로 와서 복사 붙여넣기를 하기에는 좀 힘들 수 있고
많이 쓰는 코드라면 더욱 더 함수로 바꿔야 한다.
우리가 함수를 쓰는 이유는 
하나의 기능을 함수로 구현을 해놓고
이런 코드를 계속 재사용하기 위해서이다.

함수로 바꾸면 이런식으로 사용할 수 있다.
int Factorial(int _iNum) 
{
	int iValue = 1;

	for (int j = 0; j < _iNum - 1; ++j) 
	{
		iValue *= (j + 2);
	}
	
	return iValue;
}

이런식으로 기능들을 함수로 만들어 놓는다면
코드들이 훨씬 더 간결하고 더 필요한 기능들이 많들어 져있어서
이런 모듈화 작업을 잘 해노았다면 나중에
더 복잡한 기능을 만들 때
"내가 만든 함수를 이런 식으로 합치면 되겠네"라는 경험을 할 수 있다.
이렇게 함수를 만들지 않고 그때그때 코드를 난잡하게 작성을 한다면
저 기능이 나중에 또 필요할 때 계속 또 치고 칠 수 있다.
